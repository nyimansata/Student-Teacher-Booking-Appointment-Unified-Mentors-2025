# Wireframes

This document describes the simple wireframes for the app.

1. Login / Register page

- Register fields: name, email, password, role
- Login fields: email, password

2. Student Dashboard

- Search teachers
- List of teachers with Book action
- My appointments list
- Messages

3. Teacher/Admin Dashboard

- View appointment requests
- Approve / Cancel
- Add / Edit teacher (Admin)
- Approve teacher registrations (Admin)
- Admin panel should have: Pending registrations list (approve/reject), Manage teachers (add/edit/delete), Audit logs viewer
